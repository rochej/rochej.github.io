Roche's JavaScript Prototype Cheatsheet Hover over turquoise quadrangle for code samples.

(1) Create

Write a constructor function. Pass in arguments and define THIS prototypes property names.

function dog (name, size){
this.dogName=name;
this.dogSize=size;
}

(2) Add

Add methods or properties to the prototype outside the initial definition using .prototype.

dog.prototype.dogDescriptor=""

(3) Use

Initialize a new instance of the prototype using new.

var myDog = new dog ("Misty Pepper", 32, "the cutest")
